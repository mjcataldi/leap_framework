# LEAP v1.3 Prompt Library

LEAP v1.3 introduces a reusable operational prompt library.

## Purpose

The `templates/` directory contains lightweight request templates.

The `prompts/` directory contains reusable operational prompts for:

- LEAP Recon
- LEAP Prompt generation
- strategic reconciliation
- branch drift review
- architecture governance
- cross-layer impact analysis
- multi-worktree coordination

## Directory structure

```text
prompts/
  README.md
  leap/
    recon/
      v1.3/
        leap-recon-standard.md
    implementation/
      v1.3/
        leap-prompt-standard.md
    governance/
      v1.3/
        strategic-reconciliation-pass.md
```

## Philosophy

The templates are optimized for:

- short ad-hoc requests
- quick manual invocation
- lightweight interaction

The operational prompt library is optimized for:

- repeatable workflows
- consistent governance
- architecture reconciliation
- institutional memory
- stable multi-agent execution patterns

## Recommended usage

### Use templates when:

- running a quick Recon pass
- generating a one-off implementation prompt
- experimenting
- manually driving implementation

### Use the operational prompt library when:

- running recurring workflows
- coordinating multiple worktrees or branches
- reconciling roadmap drift
- running governance reviews
- managing large layered systems
- preserving architectural consistency over time

## Relationship to LEAP v1.3

The prompt library operationalizes the strategic reconciliation additions introduced in LEAP v1.3.

The prompt files are intended to evolve independently from the framework documentation while remaining version-aligned.
